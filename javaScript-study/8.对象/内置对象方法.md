## Math对象（静态）

```js
Math.PI  圆周率
Math.random()  //生成随机数
Math.floor()   //向下取整 取的整数小于等于原来的数
Math.ceil()    //向上取整 取的整数大于等于原来的数
Math.round()   //取整,四舍五入
Math.abs()     //绝对值
Math.max()/Math.min()  //求最大值和最小值
```


## Data对象 （动态）

```js
    var time = new Data();
    time.valueOf()   //获取距1970年1月1日（世界标准时间）起的毫秒数

    getMilliseconds()  //返回0-99.
    getSeconds()       //返回0-59
    getMinutes()       //0-59
    getHours()         //0-23
    getDay()           //返回星期几  0是周日  6是周六
    getMonth()          //返回月份  从0开始 0-11
    getDate()           //返回当前月的第几天
    getFullYear()       //返回4为的年份  如2016
```
