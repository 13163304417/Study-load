#  布局

## 使用 div 元素的 HTML 布局

div 元素经常用来作布局工具

clear 属性（css中的属性）

默认值为 none

clear:left;在左侧不允许浮动元素

clear:right;在右侧不允许浮动元素

clear:both;在左右侧都不允许浮动元素

clear:inherit;规定应该从父元素继承clear属性的值

## 浮动元素的四个特点

1. 脱离文档流。就是将元素从普通的布局排版中拿走，其他荷载在定位的时候，会当做脱离文件的元素不存在而进行定位。

2. 向左/右浮动知道遇到父元素或者别的浮动元素。

3. 浮动会导致父元素高度坍塌（tanta）。

4. 浮动元素不占空间。
